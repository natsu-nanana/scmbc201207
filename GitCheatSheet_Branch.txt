
git branch ブランチ操作に関するコマンド

git branch "ブランチ名"   
  現在のブランチから新しいブランチを作る

git branch -r
  リモートのブランチ名を表示する

git branch -a
  リモートとローカルのブランチ名を表示する

git branch -m "既存のブランチ名" "新しいブランチ名"
  ブランチの移動or名前の変更



git checkout

git checkout "ブランチ名"
  指定したブランチをチェックアウトする

git checkout -b "ブランチ名"
  現在のブランチから新しいブランチを作・ﾄ、それをチェックアウトする

git commit 登録した変更内容をローカルリポジトリに反映させる。
git merge 対象のブランチの変更を、現在のブランチに取り込む。
