
git branch ブランチ操作に関するコマンド

git branch "ブランチ名"   
  現在のブランチから新しいブランチを作る

git branch -r
  リモートのブランチ名を表示する

git branch -a
  リモートとローカルのブランチ名を表示する

git branch -m "既存のブランチ名" "新しいブランチ名"
  ブランチの移動or名前の変更

git checkout

git checkout "ブランチ名"
  指定したブランチをチェックアウトする

git checkout -b "ブランチ名"
  現在のブランチから新しいブランチを作る、それをチェックアウトする

git pull originリポジトリから変更を取得して現在のブランチにマージする
git pull "リモートレポジトリ名" リモートリポジトリから変更を取得して現在のブランチにマージする
※pullは現在のブランチにマージするので使い方に注意が必要

git fetch originレポジトリの変更点を取得するが、ローカルブランチにはマージしない

git add 現在のブランチの変更点をインデックスに追加する

git commit 登録した変更内容をローカルリポジトリに反映させる。

git merge 対象のブランチの変更を、現在のブランチに取り込む。

git status 作業ツリー内のファイルの状態を表示する

git rebase 現在のブランチのHEADを<commit>の状態に戻す
